Detailed Description

This file controls what components should be debugged.

Author
Anderson Zudio (azudi.nosp@m.o@id.nosp@m..uff..nosp@m.br)

To use this, you should compile everything in DEBUG_MODE (see makefile for details). Individually uncomment lines inside this file to debug the respective component. Each component has a basic debug controller. When activated, debug code will run to help the analysis of the component with prints in std::cerr.
